flickr downloadr

A desktop application for windows that would help download all (or selected) photos from the user's photostream (in one of the selected sizes) along with the tags, titles and descriptions.

Download & Install »

Version 0.6.0.0 (beta)

Features

  • Photos downloaded in original sizes (PRO accounts)
  • Download all, all public or selected photos
  • Title, Tags, Description etc. downloaded as JSON
  • Uses OAuth to authenticate via Flickr
